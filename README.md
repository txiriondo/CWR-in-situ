# CWR in situ

A field application for the **in situ conservation of crop wild relatives (CWR)**: recording of populations within genetic reserves, with free-text species, GPS, reserve-area polygons, EUNIS habitats, UTM ETRS89 occupancy grids, occupancy frequency and transect-based abundance estimation. Works offline as an installable PWA.

## Install / host
Serve the files over HTTPS (e.g. GitHub Pages) and open `FlorCWR.html` on a phone; add it to the home screen. See the installation & user guide for details.
